Telegram Guide

To use Telegram, you need to obtain the following credentials:

Bot Token
User ID

Obtaining Bot Token

To create a Telegram bot and get its token:

  1. Open Telegram and search for @BotFather
  2. Start a chat with BotFather and send the command /newbot
  3. Follow the instructions to choose a name and username for your bot
  4. After completing the steps, BotFather will provide you with a bot token

Keep your bot token secure and never share it publicly. Anyone with access to your token can control your bot.

Finding Your User ID

To find your Telegram user ID:

  1. Search for @userinfobot on Telegram
  2. Start a conversation with the bot
  3. Send any message to the bot
  4. The bot will reply with your user information, including your ID

Telegram has a limit of approximately 30 messages per second for bots. Avoid excessive rates to prevent being banned.

Discord Guide

To use Discord, you need to create a webhook:

Webhook URL

Creating a Webhook

  1. Open your Discord server and navigate to Server Settings
  2. Click on "Integrations" in the left sidebar
  3. Click "Create Webhook" or edit an existing one
  4. Customize the webhook name and avatar if desired
  5. Copy the webhook URL - this is what you'll use in VxzSpammer

Discord rate limits webhooks to 30 requests per minute. The tool automatically handles these limits.

Webhook Options

VxzSpammer supports several Discord webhook options:

  • Username: Override the default webhook name
  • Avatar URL: Custom avatar for the webhook
  • Text-to-Speech: Have Discord read messages aloud
  • Embed Mode: Create rich embedded messages

Message Configuration

VxzSpammer provides several options to customize your messages:

Message Content

You can customize your message with the following options:

  • Message Content: The text to send (2000 character limit)
  • Image URL: Optional image to include
  • Message Count: How many messages to send (1-1000)
  • Delay: Time between messages in seconds
  • Spam Mode: Sequential, Burst, or Random timing

Spam Modes

VxzSpammer offers three different spam modes:

  • Sequential: Messages are sent one after another with the specified delay
  • Burst: All messages are sent at once (not recommended for large counts)
  • Random: Messages are sent with random intervals between 0.5x and 1.5x the delay

Use the character counter to avoid exceeding platform message limits. Different platforms have different limits:

  • Telegram: 4096 characters per message
  • Discord: 2000 characters per message

Troubleshooting

Common Issues

  • Connection failures: Check your credentials and internet connection
  • Rate limiting: Increase delay between messages
  • Message failures: Check character limits and content restrictions

Error Messages

The tool provides detailed error messages in the log console. Common errors include:

  • Invalid token: Check your Telegram bot token
  • Webhook error: Verify your Discord webhook URL
  • Rate limited: Wait before sending more messages or increase delay

Using the Log Console

The log console provides real-time feedback on your operations:

  • Info messages: General information about operations
  • Success messages: Confirmation of successful actions
  • Warning messages: Potential issues that may need attention
  • Error messages: Problems that prevented an action from completing
  • Debug messages: Detailed technical information (enable in settings)

You can filter log messages by type and export logs for further analysis.

Best Practices

Avoiding Detection

  • Start with small message counts to test
  • Use reasonable delays to avoid detection
  • Vary your message content when sending multiple messages
  • Use random intervals instead of fixed delays

Performance Optimization

  • Monitor the logs for any issues
  • Use the character counter to avoid length limits
  • Close other applications when sending large volumes of messages
  • Use a stable internet connection to prevent failures

Account Safety

  • Use separate accounts for testing and production
  • Regularly check your account status on each platform
  • Avoid sending suspicious content that might trigger filters
  • Respect platform terms of service to avoid bans

While VxzSpammer provides tools to avoid detection, no method is 100% effective. Always be prepared for the possibility of account restrictions.
